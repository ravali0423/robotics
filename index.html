Robotics Simulation Workspace

A ROS 2 Jazzy workspace with robotic hand simulations and an autonomous warehouse robot.

ROS 2 Jazzy Python 3.10+ Docker

Packages

arm_simulation

Robotic hand simulations with ASL gesture support, speech-to-gesture pipeline, and a full humanoid sign language interpreter.

  • Single & dual robotic hands
  • Humanoid sign language interpreter
  • Speech-to-gesture system
  • A–Z alphabet, 0–10 numbers, dance moves

warehouse_robot

Autonomous warehouse delivery robot with web joystick control, waypoint navigation, and mission logging.

  • Web joystick UI (browser-based)
  • Autonomous waypoint navigation
  • WASD terminal control
  • Mission log management

Setup

Docker (Mac & Windows)

The Docker image bundles ROS 2 Jazzy and a VNC server so you can view RViz2 from any platform.

docker build -t arm-sim .
docker run -it --name arm-sim -p 5900:5900 arm-sim

Then connect with VNC Viewer to localhost:5900.

Native (Linux)

rosdep install --from-paths src --ignore-src -r -y
colcon build --symlink-install
source install/setup.bash

Requirements

  • Ubuntu 24.04 LTS (native) or Docker
  • ROS 2 Jazzy
  • Python 3.10+
  • colcon, rosdep

Arm Simulation

Single Hand

5-finger robotic hand with ASL gestures and real-time finger counting.

ros2 launch arm_simulation robotic_hand.launch.py

Dual Hands

Two hands side by side. Supports combined finger counting (0–10) and two-character word spelling.

ros2 launch arm_simulation dual_hands.launch.py

Humanoid Sign Language Interpreter

Full humanoid (~1.5 m tall). Send a single /sign_command and the body animates — arms, fingers, and head simultaneously.

ros2 launch arm_simulation humanoid_robot.launch.py
# Then in a second terminal:
ros2 topic pub -1 /sign_command std_msgs/msg/String 'data: "hello"'
ros2 topic pub -1 /sign_command std_msgs/msg/String 'data: "dance_all"'

Supported: hello, a–z, 1–10, bye, yes, no, please, thanks, robot, disco, groove, and more.

Speech-to-Gesture

Translates spoken words into ASL gestures in real time. Works in text-only mode without a microphone.

ros2 launch arm_simulation speech_to_gesture.launch.py
# Text mode:
ros2 topic pub /text_to_sequence std_msgs/msg/String '{data: "cat"}' --once

Warehouse Robot

Autonomous delivery robot with a browser-based joystick, waypoint navigation, and mission logging.

# Terminal 1 — Gazebo simulation
ros2 launch warehouse_robot warehouse_with_joystick.launch.py

# Terminal 2 — web joystick
python3 src/warehouse_robot/scripts/web_joystick_controller.py
# Open http://localhost:8080/joystick.html

Autonomous navigation commands: pickup, deliver, return, mission, goto x y.
